This Macropad consists of 3 keys:

-1:TAB (To autofill commands in the Linux Terminal)
-2:Up Arrow ( Cycles backward through your command history, allowing you to quickly re-run or edit previous commands without retyping them)
-3:Ctrl+C (Immediately stops a running process or cancels the current command you are typing)

These 3 keys are supposed to help the user to navigate more easily in the Terminal
